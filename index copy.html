The History of Web Design and Development

In 1989, Tim Berners-Lee proposed to create a global hyper text project which later became known as HTML or World Wide Web. HTML standard for Hyper Text Markup Language is the language that used to create a document on World Wide Web. HTML is use for structure and layout of a web page. HTML was developed by Tim Berners Lee in 1990. The birth of-

  1. HTML 2.0 in September 1995
  2. HTML 3.2 in January 1997
  3. HTML 4.01 in December 1999
  4. HTML 5 is coming up next...

  • Coda 2, etc.

CSS

Cascading Style Sheet (CSS) is a style sheet language used for describing the presentation of a document written in a markup language. CSS is designed to enable the separation of document content from document presentation, including elements such as the layout, colors, and fonts.
